## Getting started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Requires

-   Git
-   Node

### Development environment

**1.** Install dependencies:

```bash
npm i
```

**2.** Execute:

```bash
npm start
```

**3.** Open http://localhost:3000/
